# Badge 徽章

## 概述

徽章主要用于突出显示新的或未读的项，或对某些内容进行高亮标注。

## FAQ

#### 徽章和标签的区别是什么？
答：在 Bootstrap 中，徽章和标签类似，区别只是徽章的边角更加圆滑，在 Shine Design 中，我们将徽章和标签进行了整合，标签的应用场景均可以用徽章替代。

#### 徽章可以用在哪些地方？
答：徽章组件本身是一个基础组件，它可以被嵌入到任何想展现的位置，例如菜单项 /  导航栏 等，也可以作为状态展示，例如用户登录状态（在线 / 离线）。
