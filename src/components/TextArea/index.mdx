# TextArea 文本域

## 概述
文本域组件是一种可容纳多行文本的文本域组件，其中的文本的默认字体是等宽字体。

## 代码示例

### 文本域类型

使用 `textAreaStyle` 属性可以指定单个文本域的类型，目前支持三种类型，通过不同类型搭配可以实现个性化表单。
