# Checkbox 单选框

## 概述
在页面上生成一个表单容器，用户可以通过鼠标、键盘等设备输入内容，是最基本的表单组件。

## 代码示例

#### 基础单选框

输入框组件是由基础输入标签封装而成，可以快速的创建一些常见的表单控件。
