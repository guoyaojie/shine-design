# Input 输入框

## 概述
在页面上生成一个表单容器，用户可以通过鼠标、键盘等设备输入内容，是最基本的表单组件。

## 代码示例

使用 `formStyle` 属性可以指定单个输入框的类型，目前支持三种类型，通过不同类型搭配可以实现个性化表单。
