# 更新日志

## 发布周期
* 修订版本号：每周末会进行日常 bugfix 更新。（如果有紧急的 bugfix，则任何时候都可发布）
* 次版本号：每月发布一个带有新特性的向下兼容的版本。
* 主版本号：含有破坏性更新和新特性，不在发布周期内。

---

## 0.2.7
2018-11-20 发布
### 新特性
* 发布表单容器组件： `Form` 
* 表单类组件支持配置表单校验

* 修复基础输入组件 `Input` 组件在 type 类型为 checkbox 或 radio 时 id、name 属性重复问题
* 修复表单类组件在 inline 模式下排版问题
* 修复表单文字显示异常问题
