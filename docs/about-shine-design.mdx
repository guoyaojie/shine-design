# Shine Design
为开发者、设计师和产品经理准备的 UI 设计语言

## 特性

* 开箱即用的高质量组件，为开发者、设计师和产品经理提供丰富的创意和思路。
* 兼容 IE, Chrome 和 Firefox 等绝大部分主流浏览器，为用户带来统一的视觉体验。
* 拥有全链路开发和设计工具体系，永久免费。
